# HOMEHUNTERS
** HomeHunterconnects home finders to selection of accommodations. At the same time, it allows homeowners to market their property. **

## Terminal 1: Frontend
```
cd frontend
npm run start
```

## Terminal 2: Backend
```
cd backend
npm run start
```
